# 🚴 Bike Sharing Data Analysis

##  Project Overview
This project explores a bike-sharing dataset to understand user behavior and trip patterns.

The analysis is divided into two parts:
- Exploratory Data Analysis (EDA)
- Explanatory Data Analysis (Storytelling)

---

## 🔍 Key Questions
- How are trip durations distributed?
- How do Subscribers and Customers differ?
- What patterns exist across time and age?

---

## 📈 Key Insights
- Most trips are short (right-skewed distribution)
- Customers take longer trips than Subscribers
- Subscribers show strong commute patterns (morning & evening)
- Age has very little impact on trip duration
- User type is the most important factor

---

## 🧰 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Files
- `Part_I_exploration.ipynb` → exploratory analysis
- `Part_II_explanatory.ipynb` → storytelling & insights

---

## 📌 Conclusion
Subscribers mainly use the system for daily commuting, while Customers use it more for leisure activities.

---

## 📎 Dataset
Ford GoBike System Data (2019)
